# 用語集

このドキュメントでは、ローカルメールアプリケーションのドキュメントで使用される重要な用語と略語を定義します。

## A

### App Router（アプリケーションローター）
SAPアプリケーションローターは、リクエストをマイクロサービスにルーティングし、認証を処理するNode.jsアプリケーションです。

## C

### CAP（Cloud Application Programming Model）
SAPのフレームワークであり、エンタープライズグレードのサービスとアプリケーションを構築するための言語、ライブラリ、ツールを提供します。

### CDS（Core Data Services）
SAPが定義したデータ定義言語。エンティティ、リレーションシップ、サービスを定義するために使用されます。

### Cloud Foundry
SAP BTPで使用されるオープンソースのクラウドアプリケーションプラットフォーム。

### CRUD
Create、Read、Update、Delete - データに対して実行できる4つの基本操作。

## D

### Draft（ドラフト）
Fiori Elementsの機能で、ユーザーが変更を確定する前に、ユーザーがエンティティのドラフトバージョンを作成できるようにします。

## F

### Fiori Elements
SAPのUI開発フレームワークで、注釈に基づいて標準化されたUIパターンを提供します。

### FLP（Fiori Launchpad）
SAP Fioriアプリケーションへのエントリーポイントを提供するロールベースのランチパッド。

## H

### HANA Cloud
SAPのクラウドネイティブインメモリデータベース。

## J

### JWT（JSON Web Token）
当事者間でクレームを安全に送信するためのコンパクトでURLセーフな手段。

## L

### List Report
Fiori Elementsパターンで、フィルタ可能なテーブルでエンティティのリストを表示します。

## M

### Managed Aspect
`createdAt`、`createdBy`、`modifiedAt`、`modifiedBy`フィールドを自動的に提供するCAPの組み込みアスペクト。

### MTA（Multi-Target Application）
マルチターゲットアプリケーション - SAP BTPでのデプロイメント用に設計されたアプリケーションパッケージ。

## O

### OData
Open Data Protocol - データの作成、読み取り、更新、削除に使用されるOASIS標準。

### Object Page
Fiori Elementsパターンで、単一のエンティティの詳細な情報を表示します。

### OAuth 2.0
認可のための業界標準プロトコル。

## R

### RBAC（Role-Based Access Control）
ロールベースアクセス制御 - ロールをユーザーに割り当てて権限を決定するセキュリティアプローチ。

### RDL（Runtime Definition Language）
ランタイム定義言語。

## S

### SAPUI5
SAPのHTML5フレームワークで、エンタープライズグレードのWebアプリケーションを構築するために使用されます。

### Service（サービス）
HTTPプロトコルを使用してアクセスできる機能セットを公開するソフトウェア。

### Scope（スコープ）
OAuthの概念で、アプリケーションに付与された特定の権限を定義します。

## U

### UI5
SAPUI5またはOpenUI5の短縮形 - SAPのフロントエンドフレームワーク。

### UUID
Universally Unique Identifier - 一意の識別子。

## X

### XSUAA
Extended Services for User Account and Authentication - SAP BTPの認証サービス。
